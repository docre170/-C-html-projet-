Comment installer Trebedit ?

C'est quoi un éditeur de code

Un éditeur de code est un logiciel conçu pour faciliter l'écriture et la modification de code source dans divers langages de programmation. Il offre des fonctionnalités qui aident les développeurs à coder plus efficacement. Voici quelques caractéristiques typiques d'un éditeur de code :

  1. Coloration syntaxique : L'éditeur met en évidence les différentes parties du code (comme les mots-clés, les variables, les commentaires, etc.) avec des couleurs différentes pour améliorer la lisibilité.
  2. Complétion automatique : Il propose des suggestions de code pendant que vous tapez, ce qui peut accélérer le processus d'écriture.
  3. Indentation et mise en forme : L'éditeur aide à organiser le code en appliquant automatiquement des règles d'indentation et de mise en forme.
  4. Gestion des fichiers : Il permet d'ouvrir, d'enregistrer et de gérer plusieurs fichiers de code dans un projet.
  5. Intégration avec des systèmes de contrôle de version : Certains éditeurs permettent d'intégrer des outils comme Git pour gérer les versions du code.
  6. Extensions et plugins : De nombreux éditeurs de code peuvent être personnalisés avec des extensions pour ajouter des fonctionnalités supplémentaires, comme le support pour des langages supplémentaires ou des outils de débogage.

Quel éditeur utiliser ?

Il existe plusieurs éditeurs de code plus précisément pour HTML sur Android comme : HTML Creator, Dcoder, Pydroid3, HTML Creator, etc. Mais nous allons utiliser l'éditeur de code Trebedit.

Comment installer Trebedit ?

Installer Trebedit est plutôt facile. Il suffit juste de suivre les instructions ci-dessous :

  • Ouvrez d'abord Playstore ou Palmstore.
  • Allez dans recherche.
  • Dans la barre de recherche : recherchez Trebedit.
  • Cliquez sur n'importe quel Trebedit.
  • Une fois le téléchargement terminé, cliquez sur ouvrir.
  • Une fois dans Trebedit, cliquez sur menu.
  • Puis choisissez éditeur de texte.
  • Une fois à l'intérieur, cliquez sur l'icône dossier.
  • Recherche vous permettra de trouver du code; sauvegarder vous permettra d'enregistrer du code; lecture vous permettra de lire les codes édités.
  • Cliquez sur +.
  • Cliquez sur créer un nouveau fichier.
  • Espace de travail vous permettra de choisir où le fichier créé sera conservé; nom fichier vous permettra de nommer le fichier.
  • Vous pouvez écrire n'importe quel nom ex : nom.html. Le .HTML permettra à l'éditeur de reconnaître que c'est du code HTML.
  • Cliquez sur sélecteur de fichiers de périphérique : pour choisir le dossier où le fichier sera gardé.
  • Choisissez un dossier (de préférence créez un nouveau dossier).
  • Là cliquez sur utiliser ce dossier.
  • Cliquez sur autoriser.
  • Puis sauvegarder.
  • Cliquez sur extrait de code HTML.
