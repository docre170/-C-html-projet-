Module 2 : Bases de données et tables

1. Qu’est-ce qu’une base de données ?

Une base de données (BDD) est un ensemble organisé de données stockées et accessibles électroniquement. Elle permet de :
- Stocker des informations de manière structurée.
- Manipuler (ajouter, modifier, supprimer) ces données.
- Rechercher efficacement des informations.

2. Qu’est-ce qu’une table ?

Une table est une structure principale dans une base de données, organisée en lignes (enregistrements) et colonnes (champs).
- Colonne (Champ) : Représente un type de donnée (ex: Nom, Âge, Email).
- Ligne (Enregistrement) : Une entrée complète dans la table (ex: "Jean", 25, "jean@exemple.com").

3. Clé primaire (Primary Key - PK)

- Définition : Une colonne (ou un ensemble de colonnes) qui identifie de manière unique chaque ligne.
- Exemple : Un ID unique pour chaque utilisateur.

4. Types de données courants dans une table

- INT : Nombre entier (ex: âge).
- VARCHAR : Texte de longueur variable (ex: nom, email).
- DATE : Date (ex: date de naissance).
- BOOLEAN : Vrai/Faux (ex: actif/inactif).

En résumé

- Une base de données stocke des données de manière organisée.
- Une table est une structure avec des lignes et colonnes.
- La clé primaire assure l'unicité des enregistrements.
- Les relations permettent de lier plusieurs tables.
- Les opérations CRUD permettent de manipuler les données.
