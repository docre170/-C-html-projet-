Voici un aperçu clair et rapide des bases de Python:

1. Variables et types de données

Une variable est une boîte dans laquelle on stocke une valeur.

```python
nom = "Alice"        # chaîne de caractères (str)
âge = 25             # entier (int)
taille = 1.68        # nombre décimal (float)
étudiant = True      # booléen (bool)
```

2. Fonctions `input()` et `print()`

- `input()`: demande à l'utilisateur d'entrer une valeur.

- `print()`: affiche un message à l'écran.

```python
nom = input("Quel est ton nom? ")
print("Bonjour, " + nom)
```

3. Opérateurs arithmétiques

Ils permettent de faire des calculs:

```python
+   Addition        3 + 2 → 5
-   Soustraction    5 - 1 → 4
*   Multiplication  4 * 3 → 12
/   Division        6 / 2 → 3.0
//  Division entière 7 // 2 → 3
%   Modulo (reste)  7 % 2 → 1
**  Puissance       2 ** 3 → 8
```

4. Opérateurs logiques

Ils comparent des valeurs (vrai ou faux):

```python
x = 10
y = 5

print(x > y)    # True
print(x == y)   # False
print(x != y)   # True
```

Opérateurs:
- `==` égal
- `!=` différent
- `>` supérieur
- `<` inférieur
- `>=` supérieur ou égal
- `<=` inférieur ou égal
