1. Conditions (`if`, `elif`, `else`)

Les conditions permettent d'exécuter un bloc de code uniquement si une condition est vraie.

```python
âge = 18

if âge >= 18:
    print("Adulte")
elif âge >= 13:
    print("Adolescent")
else:
    print("Enfant")
```

2. Boucles (`for` et `while`)

Les boucles permettent de répéter un bloc de code plusieurs fois.

`for`: Parcourt une séquence

```python
for i in range(5):  # De 0 à 4
    print(i)
```

`while`: Répète tant que la condition est vraie

```python
x = 0
while x < 5:
    print(x)
    x += 1
```

3. Instrumentation (`break` et `continue`)

Ces mots-clés permettent de contrôler l'exécution des boucles.

`break`: Arrête la boucle immédiatement

```python
for i in range(10):
    if i == 5:
        break  # Sort de la boucle
    print(i)
```

continue: Passe à l'itération suivante sans exécuter le reste du bloc

```python
for i in range(5):
    if i == 2:
        continue  # Ignore le 2
    print(i)
```
