Module 2 : Représentation des données

Définition

Représenter une donnée, c’est montrer ou stocker une information d’une façon que l’ordinateur peut comprendre.
Un ordinateur ne voit pas une image ou une musique comme nous. Il la représente avec des nombres.

exemples :

1. Texte (les lettres)

- Chaque lettre a un code numérique.
- Exemple :
- A = 65
- B = 66
- En binaire : A = 01000001

2. Image

  • - Une image est faite de petits carrés appelés pixels.
  • - Chaque pixel a une couleur, et chaque couleur est codée par des nombres (souvent en RGB : Rouge, Vert, Bleu).
    - Exemple : Rouge pur = (255, 0, 0)
L’ordinateur enregistre l’image comme une liste de codes de couleurs.

3. Son

- Le son est une vibration. L’ordinateur le mesure à plusieurs moments.
- Chaque point mesuré est un nombre.
- Plus il y a de points (fréquence d’échantillonnage), plus le son est clair.
Un fichier audio est une longue liste de chiffres représentant le son.

4. Vidéo

- Une vidéo = plein d’images (comme un dessin animé).
- Chaque image est une image codée + un son codé.

Pourquoi c’est important ?

Pour que l’ordinateur puisse comprendre toutes les choses que nous utilisons chaque jour (photos, musique, documents…).
Pour transmettre, afficher, ou modifier correctement les données.
