Introduction au CSS

CSS, c'est quoi ?

CSS signifie Cascading Style Sheets ou Feuilles de style en cascade. C'est un langage qui sert à décorer tes pages HTML. Grâce à lui, tu peux changer :

  • les couleurs
  • les tailles de texte
  • les espaces
  • les positions
  • les polices
  • et bien plus

Petite histoire rapide

À la base, le HTML ne servait qu'à structurer le contenu (titres, paragraphes, images...). Mais tout était très moche.

Alors, en 1996, le CSS est né pour séparer le contenu (HTML) de la présentation (style).

Pourquoi c'est important ?

Tu rends ton site beau et agréable.

  • Tu gagnes du temps : un seul fichier CSS peut styliser plusieurs pages
  • Tu fais des sites professionnels et responsives

Exemple très simple

<html>
  <h1>Bienvenue</h1>
  <p>CSS</p>
</html>

h1 {
  color: blue;
  font-size: 36px;
}

Ça transforme le titre en gros texte bleu stylé !
