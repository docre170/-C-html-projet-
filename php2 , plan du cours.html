Plan de cours – Langage PHP (Niveau débutant)

Module 1 : Introduction à PHP

  • Qu’est-ce que PHP ?
  • Fonctionnement côté serveur
  • Installer un environnement local (XAMPP, WAMP, etc.)
  • Premier script PHP : `echo`, `print`

Module 2 : Bases du langage

  • Variables et types de données
  • Opérateurs arithmétiques et logiques
  • Structures conditionnelles (`if`, `else`, `elseif`, `switch`)
  • Boucles (`while`, `for`, `foreach`)

Module 3 : Fonctions

  • Déclaration et appel de fonctions
  • Passage de paramètres
  • Valeurs de retour

Module 4 : Tableaux

  • Tableaux indexés
  • Tableaux associatifs
  • Parcourir un tableau avec `foreach

Module 5 : Formulaires et PHP

  • Récupération de données avec `_GET` et `_POST`
  • Validation simple de formulaire
  • Affichage des résultats

Module 6 : Fichiers

  • Lire et écrire dans un fichier (`fopen`, `fwrite`, `fread`)
  • Télécharger un fichier

Module 7 : Introduction à MySQL avec PHP

  • Connexion à une base de données avec `mysqli`
  • Exécution de requêtes : `SELECT`, `INSERT`
  • Affichage de résultats dynamiques

Module 8 : Sécurité de base

  • Échapper les entrées utilisateur
  • Introduction à `htmlspecialchars()`, `mysqli_real_escape_string()`
