Module 3 : Fonctions en PHP

Les fonctions sont des éléments fondamentaux en PHP qui permettent d'organiser et de réutiliser du code. Voici les concepts clés de ce module :

1. Déclaration de fonctions 

                    ```php
function nomFonction($param1, $param2 = "valeur par défaut") {
    // Corps de la fonction
    return $resultat; // Optionnel
}
```

2. Types de fonctions

Fonctions simples 

                    ```php
function saluer() {
    echo "Bonjour !";
}
```

Fonctions avec paramètres 

                    ```php
function additionner($a, $b) {
    return $a + $b;
}
```

Fonctions avec valeurs par défaut 

                    ```php
function direBonjour($nom = "invité") {
    echo "Bonjour, $nom !";
}
```

3. Retour de valeurs

- Utilisation du mot-clé `return`
- Une fonction peut retourner n'importe quel type de données
- Retourne `null` si aucun return n'est spécifié

4. Portée des variables

- Les variables déclarées dans une fonction sont "locales" - Pour accéder à une variable globale : 

                    ```php
$globale = 10;

function test() {
    global $globale;
    echo $globale;
}
```

5. Fonctions variables 

                    ```php
function maFonction() {
    echo "Fonction appelée";
}

$fonction = "maFonction";
$fonction(); // Appelle maFonction()
```

6. Fonctions anonymes (closures) 

```php
$addition = function($a, $b) {
    return $a + $b;
};

echo $addition(2, 3); // Affiche 5
```

. Typage des paramètres et retours (PHP 7+) 

                    ```php
function calculerMoyenne(float ...$nombres): float {
    return array_sum($nombres) / count($nombres);
}
```

8. Fonctions récursives 

                    ```php
function factorielle($n) {
    if ($n <= 1) return 1;
    return $n * factorielle($n - 1);
}
```

Bonnes pratiques

  1. Donner des noms explicites aux fonctions
  2. Limiter la taille des fonctions (une fonction = une tâche)
  3. Documenter avec des commentaires PHPDoc
  4. Utiliser des paramètres avec des valeurs par défaut judicieusement
  5. Éviter les effets de bord (modification de variables globales)

Fonctions utiles intégrées

PHP propose de nombreuses fonctions prédéfinies :

  • `strlen()`, `substr()` pour les chaînes
  • `count()` pour les tableaux
  • `date()` pour les dates
  • `file_get_contents()` pour la lecture de fichiers
  • `json_encode()`/`json_decode()` pour le JSON
