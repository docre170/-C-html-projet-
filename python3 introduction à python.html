Qu'est-ce que Python?

Python est un langage de programmation très populaire pour sa syntaxe simple et lisible. Il est utilisé dans de nombreux domaines: développement web, intelligence artificielle, analyse de données, automatisation, jeux vidéo, etc.

Pourquoi on l'aime:

  • Facile à apprendre, même pour les débutants.
  • Une communauté énorme et beaucoup de ressources gratuites.
  • Des milliers de bibliothèques pour t'aider à tout faire plus rapidement.

Comment installer Python?

1. Va sur le site officiel:
lien pour télécharger python

2. Clique sur le bouton Download Python (la version la plus récente est recommandée).

3. Lance le fichier téléchargé.

4. Important: coche la case "Add Python to PATH" avant de cliquer sur "Install Now".

Une fois installé, tu peux vérifier que tout fonctionne:
- Ouvre un terminal (ou l'invite de commande).
- Tape: 

python --version

Ton tout premier script Python

1. Ouvre un éditeur de texte simple comme le Bloc-notes (ou un IDE, on y revient juste après).
2. Tape ce petit code:

print("Bonjour, Python!")
3. Sauvegarde le fichier avec l'extension .py, par exemple hello.py.
4. Dans ton terminal, exécute-le avec: 
python hello.py
Et voilà, ton premier programme Python!

Utiliser un IDE (Environnement de Développement Intégré)

Un IDE est un outil qui te simplifie la vie pour coder. Il t'aide avec:
- La coloration syntaxique
- L'auto-complétion du code
- Le débogage

Recommandé pour débuter: Thonny (simple et conçu pour apprendre) ou VS Code.

1. Télécharge ici:
télécharger thonny
2. Installe-le et ouvre-le.
3. Tu peux y écrire directement ton script, puis cliquer sur "Run".

Autres IDE populaires (un peu plus avancés): VS Code, PyCharm, Jupyter Notebook (super pour les datas).

Nous espérons que vous avez compris ce cours car l'aventure vient de commencer!
