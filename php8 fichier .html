1. Ouvrir un fichier – `fopen()`

- Sert à ouvrir un fichier pour lire ou écrire.

Syntaxe :
`fopen("nom_du_fichier.txt", "mode");`

Exemples de modes :

  • `"r"` : lecture seule
  • `"w"` : écriture seule (efface le contenu si le fichier existe)
  • `"a"` : écriture en ajoutant à la fin
  • `"r+"` : lecture et écriture
```php
$fichier = fopen("test.txt", "w");
```

2. Écrire dans un fichier – `fwrite()`

- Sert à écrire du texte dans un fichier ouvert en mode écriture. 

```php
fwrite($fichier, "Bonjour le monde !");
fclose($fichier);
```

3. Lire un fichier – `fread()`

- Sert à lire le contenu d'un fichier. 

```php
$fichier = fopen("test.txt", "r");
$contenu = fread($fichier, filesize("test.txt"));
fclose($fichier);
echo $contenu;
```

4. Télécharger un fichier avec PHP

```php
header('Content-Type: application/octet-stream');
header('Content-Disposition: attachment; filename="fichier.txt"');
readfile("chemin/fichier.txt");
exit;
```
