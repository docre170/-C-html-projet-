Fonction

On utilise `def` pour créer une fonction.

```python
def saluer():
    print("Bonjour!")
```

Appel de la fonction: `saluer()`

2. Paramètres

On peut passer des valeurs à une fonction via les paramètres :

```python
def bonjour(nom):
    print("Salut, " + nom)

bonjour("Alice")
```

3. Valeur de retour avec `return`*
Une fonction peut *renvoyer* une valeur:

```python
def addition(a, b):
    return a + b

résultat = addition(3, 5)
print(résultat)  # Affiche 8
```

4. Portée des variables

Il y a 2 types de portée:
- *Locale*: la variable est accessible _seulement dans la fonction_.
- *Globale*: la variable est accessible _partout_ dans le programme.

```python
x = 10  # variable globale

def test():
    x = 5  # variable locale (différente)
    print(x)

test()       # Affiche 5
print(x)     # Affiche 10
```
