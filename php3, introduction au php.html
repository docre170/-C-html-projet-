Qu'est-ce que PHP ?

PHP, c'est un langage magique qui aide les sites web à réfléchir et à réagir.

Par exemple, si tu vas sur un site pour te connecter avec ton prénom et ton mot de passe, c'est PHP qui vérifie si c'est correct.

À quoi sert PHP ?

Imagine un site web comme un robot. HTML et CSS font que le robot est beau et bien habillé. Mais PHP, c'est ce qui lui donne un cerveau.

Avec PHP, on peut faire :
Des formulaires pour s'inscrire ou se connecter

Des jeux simples
Des sites où tu peux voir ton nom, tes messages, ton profil
Des boutiques en ligne

Comment ça marche ?

PHP travaille dans les coulisses. Quand tu vas sur un site, PHP lit ce que tu veux, fait des calculs si besoin, et t'envoie une page avec la bonne réponse.

Par exemple : Tu écris ton nom → PHP le lit → il dit : « Bonjour, [Ton prénom] ! »

À quoi ressemble PHP ?

Voici une petite phrase en PHP : 

<?php
echo "Bonjour !";
?>

Le mot echo veut dire "dis quelque chose".
Donc cette phrase dit : "Bonjour !"

Pour utiliser PHP, il faut :

1. Un ordinateur ou un serveur qui sait lire le PHP (comme un prof qui comprend cette langue).

2. Un navigateur (comme Chrome) pour voir le résultat.

3. Écrire ton code PHP entre <?php et ?>

Résumé

PHP, c'est un langage qui aide les sites à penser.
Il est très utile pour faire des sites intelligents.
Tu peux écrire du PHP pour faire parler ton site, répondre aux gens, ou garder des infos.
