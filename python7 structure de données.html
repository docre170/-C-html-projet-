1. Liste (`list`)

Une liste est une collection modifiable et ordonnée. 

```python
fruits = ["pomme", "banane", "cerise"]
print(fruits[0])  # pomme
fruits.append("mangue")  # ajoute à la fin
```

2. Dictionnaire (`dict`)

Un dictionnaire contient des paires clé-valeur, comme un mini-annuaire. 

```python
personne = {"nom": "Alice", "âge": 25}
print(personne["nom"])  # Alice
personne["ville"] = "Paris"  # ajout d'une nouvelle clé
```

3. Tuple (`tuple`)

Un tuple est comme une liste mais immuable (on ne peut pas le modifier). 

```python
coordonnées = (10.5, 42.3)
print(coordonnées[1])  # 42.3
```

4. Ensemble (`set`)

Un ensemble contient des éléments uniques, sans ordre. 

```python
couleurs = {"rouge", "bleu", "vert"}
couleurs.add("jaune")
```
Les ensembles sont parfaits pour éliminer les doublons.
