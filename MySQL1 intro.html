Module 1 : Introduction à MySQL

1. Qu'est-ce que MySQL ?

MySQL est un système de gestion de bases de données relationnelles (SGBDR) open-source. Il permet de :
- stocker des données de manière organisée.
- Manipuler ces données (ajout, modification, suppression).
- Interroger les données avec le langage SQL (Structured Query Language).

2. Pourquoi utiliser MySQL ?

- Gratuit et Open Source : Disponible sous licence GPL.
- Performant : Optimisé pour les applications web et les grosses charges.
- Sécurisé : Gestion des utilisateurs et des permissions.
- Multiplateforme : Fonctionne sur Windows, Linux, macOS.
- Compatible avec de nombreux langages (PHP, Python, Java, etc.).

3. Outils pour utiliser MySQL

- MySQL Workbench (Interface graphique).
- phpMyAdmin (Gestion via navigateur web).
- Ligne de commande (Terminal MySQL).

Conclusion

MySQL est un SGBDR puissant et largement utilisé pour stocker et gérer des données structurées. La maîtrise des bases (tables, requêtes SQL) est essentielle pour développer des applications web ou logicielles.
