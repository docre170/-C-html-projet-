Module 2 : Bases du langage en PHP

Le Module 2 couvre les fondamentaux du langage PHP, essentiels pour commencer à développer des applications web. Voici les principales notions abordées :

1. Syntaxe de base

- Structure d'un script PHP : <?php ... ?>
- Instruction de fin : point-virgule (;)
- Commentaires : // pour une ligne, /* ... */ pour un bloc

2. Variables et types de données

- Déclaration : $nomVariable = valeur;
- Types principaux :
- Chaînes (string) : $texte = "Bonjour";
- Entiers (int) : $nombre = 42;
- Nombres à virgule (float) : $decimal = 3.14;
- Booléens (bool) : $vrai = true;
- Tableaux (array) : $liste = [1, 2, 3];
- NULL

3. Opérateurs

- Arithmétiques : `+`, `-`, `*`, `/`, `%` (modulo)
- Comparaison : `==`, `===` (égalité stricte), `!=`, `!==`, `>`, `>`, etc.
- Logiques : `&&` (et), `||` (ou), `!` (non)
- Concaténation de chaînes : `.`

4. Structures de contrôle

- Conditionnelles : 

                   ```php
  if (condition) {
      // code
  } elseif (autre_condition) {
      // code
  } else {
      // code
  }
  ```

- Boucles :

  ```php
  for ($i = 0; $i < 10; $i++) { /* code */ }
  
  while (condition) { /* code */ }
  
  do { /* code */ } while (condition);
  
  foreach ($tableau as $element) { /* code */ }
  ```

5. Fonctions

Définition : 

                     ```php
  function maFonction($param1, $param2) {
      // code
      return $resultat;
  }
  ```

- Appel : `maFonction($arg1, $arg2);`

6. Tableaux

- Indexés : 

                   ```php
  $fruits = ["pomme", "banane", "orange"];
  echo $fruits[0]; // "pomme"
  ```

- Associatifs : 

                   ```php
  $personne = [
      "nom" => "kutanga",
      "prenom" => "credo",
      "age" => 25
  ];
  echo $personne["nom"]; // "kutanga"
  ```

7. Superglobales

Variables prédéfinies accessibles partout :

  • - `$_GET` : paramètres URL
  • - `$_POST` : données formulaire
  • - `$_SESSION` : variables de session
  • - `$_COOKIE` : cookies
  • - `$_SERVER` : informations serveur

8. Traitement des formulaires

- Récupération des données : 

                   ```php
  $nom = $_POST['nom']; // pour un formulaire en POST
  $recherche = $_GET['q']; // pour des paramètres URL
  ```

Bonnes pratiques

- Validation des entrées utilisateur
- Protection contre les injections SQL
- Utilisation de `htmlspecialchars()` pour l'affichage
