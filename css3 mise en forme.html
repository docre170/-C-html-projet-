Sélecteurs et propriétés

Imagine que tu as une boîte magique pleine de vêtements. Chaque vêtement a une couleur différente, une taille différente, et parfois même des motifs rigolos. Maintenant, imagine que tu veux dire à la boîte magique: "Hé, mets-moi mon t-shirt rouge!" ou "Change la couleur de mes chaussettes!"

Dans le monde des sites web, ces vêtements représentent des éléments d'une page web comme des titres, des images ou des boutons. Pour parler à la boîte magique (qui est comme le langage CSS), tu dois utiliser des sélecteurs et des propriétés.

1. Les Sélecteurs

Les sélecteurs, c'est comme pointer du doigt un vêtement précis et dire: "C'est celui-ci que je veux changer!" Il existe plusieurs manières de choisir un élément:

  • Le sélecteur de type (ex: p pour les paragraphes, h1 pour les titres)
  • Le sélecteur de classe (ex: .important pour tous les éléments ayant cette classe)
  • Le sélecteur d'identifiant (ex: #bouton pour un élément unique avec cet ID)

Exemple: Si tu veux colorer tous les pantalons de ton placard en bleu, tu utiliserais quelque chose comme: 

pantalon {
  couleur: bleu;
}

2. Les Propriétés

Les propriétés, c'est ce que tu veux modifier sur l'élément. Un peu comme dire: "Je veux que mon t-shirt soit plus grand!" ou "Mon chapeau doit être jaune!"

Voici quelques propriétés CSS courantes:

  • Couleur (color) → Change la couleur du texte.
  • Taille (font-size) → Agrandit ou réduit le texte.
  • Fond (background-color) → Change la couleur derrière un élément.
  • Marges (margin) → Éloigne les éléments les uns des autres.

Exemple: Si tu veux que tous les titres soient rouges et très gros, tu écrirais: 

h1 {
  color: red;
  font-size: 50px;
}

Voilà, CSS c'est un peu comme donner des instructions à une boîte magique qui habille une page web. Tu choisis les bons vêtements (sélecteurs) et tu leur dis quoi changer (propriétés).
