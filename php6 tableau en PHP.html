Module 4 : Tableaux en PHP

Les tableaux (arrays en anglais) sont des structures de données fondamentales en PHP qui permettent de stocker plusieurs valeurs dans une seule variable.

1. Les types de tableaux en PHP

a) Tableaux indexés:

Les tableaux indexés utilisent des indices numériques (commençant à 0 par défaut). 

$fruits = array("Pomme", "Banane", "Orange");
// ou depuis PHP 5.4+
$fruits = ["Pomme", "Banane", "Orange"];

// Accès à un élément
echo $fruits[1]; // Affiche "Banane"

b) Tableaux associatifs

Les tableaux associatifs utilisent des clés nommées (chaînes de caractères). 

$personne = array(
    "nom" => "Dupont",
    "prenom" => "Jean",
    "age" => 30
);
// ou
$personne = [
    "nom" => "Dupont",
    "prenom" => "Jean",
    "age" => 30
];

echo $personne["prenom"]; // Affiche "Jean"

c) Tableaux multidimensionnels

Ce sont des tableaux qui contiennent d'autres tableaux. 

$etudiants = array(
    array("nom" => "Martin", "notes" => [12, 15, 18]),
    array("nom" => "Durand", "notes" => [10, 14, 16])
);

echo $etudiants[0]["nom"]; // Affiche "Martin"
echo $etudiants[1]["notes"][2]; // Affiche 16

2. Manipulation des tableaux

Fonctions utiles pour les tableaux

- count() : Compter le nombre d'éléments

echo count($fruits); // 3

- array_push() / array_pop() : Ajouter/supprimer en fin de tableau

array_push($fruits, "Kiwi");
$dernier = array_pop($fruits);

- array_merge() : Fusionner des tableaux

$tableau1 = ["a", "b"];
$tableau2 = ["c", "d"];
$resultat = array_merge($tableau1, $tableau2);

- sort(), rsort(), asort(), ksort() : Trier des tableaux

sort($fruits); // tri par valeurs (index numérique)
asort($personne); // tri par valeurs en gardant les clés
ksort($personne); // tri par clés

- array_key_exists() : Vérifier si une clé existe

if (array_key_exists("age", $personne)) {
    echo "La clé age existe";
}

- in_array() : Vérifier si une valeur existe

if (in_array("Pomme", $fruits)) {
    echo "Pomme existe dans le tableau";
}

3. Parcourir un tableau

Avec for (pour les tableaux indexés)

for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "
";
}

Avec foreach (pour tous les tableaux)

foreach ($fruits as $fruit) {
    echo $fruit . "
";
}

foreach ($personne as $cle => $valeur) {
    echo "$cle : $valeur
";
}

4. Tableaux et formulaires HTML

PHP permet de récupérer des données de formulaires sous forme de tableaux : 

<form method="post">
    <input type="text" name="personne[nom]">
    <input type="text" name="personne[prenom]">
    <input type="submit">
</form>

En PHP :

$donnees = $_POST['personne'];
echo $donnees['nom']; // Affiche la valeur du champ nom

5. Bonnes pratiques

  1. Utilisez des tableaux associatifs pour plus de clarté quand c'est possible
  2. Préférez la syntaxe courte `[]` à `array()` (depuis PHP 5.4)
  3. Utilisez `foreach` pour parcourir les tableaux associatifs
  4. Vérifiez toujours si une clé existe avant d'y accéder

Les tableaux sont extrêmement puissants en PHP et constituent l'une des fonctionnalités les plus utilisées dans le langage.
