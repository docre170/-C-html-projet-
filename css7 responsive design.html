Responsive Design

Le Responsive Design (design adaptatif) permet à votre site web de s'adapter à toutes les tailles d'écran (smartphone, tablette, ordinateur).

Les Media Queries sont des règles CSS qui disent : "Si l'écran fait telle taille, alors applique ce style !"

1. Pourquoi c'est important ?

- Sur mobile, un site mal adapté oblige l'utilisateur à zoomer/dézoomer.
- Sur grand écran, le contenu peut paraître trop étiré.
- Google favorise les sites mobile-friendly dans ses résultats.

2. Comment ça marche ?

Exemple 1 : Changer la couleur sur mobile

/* Style par défaut (pour tous les écrans) */
body {
  background-color: lightblue;
}

/* Si l'écran fait moins de 600px (smartphone) */
@media (max-width: 600px) {
  body {
    background-color: pink;
  }
}

Résultat:
- Sur ordinateur → Fond bleu clair
- Sur téléphone → Fond rose

Exemple 2 : Cacher un élément sur mobile

/* Par défaut, le menu est visible */
.menu {
  display: block;
}

/* Sur mobile, on le cache */
@media (max-width: 600px) {
  .menu {
    display: none;
  }
}

Résultat :
- Sur mobile → Le menu disparaît.

Exemple 3 : Changer la taille du texte

/* Taille normale sur ordinateur */
h1 {
  font-size: 32px;
}

/* Taille réduite sur mobile */
@media (max-width: 600px) {
  h1 {
    font-size: 24px;
  }
}

Résultat :
- Sur mobile → Le titre est plus petit.

3. Points de rupture courants (Breakpoints)

On utilise souvent ces tailles d'écran :

Taille (max-width) Type d'appareil
600px Smartphones
768px Tablettes
992px Petits ordinateurs
1200px Grands écrans

Exemple d'utilisation:

/* Style pour mobile */
@media (max-width: 600px) {
  /* Règles CSS */
}

/* Style pour tablette */
@media (min-width: 601px) and (max-width: 768px) {
  /* Règles CSS */
}

/* Style pour ordinateur */
@media (min-width: 769px) {
  /* Règles CSS */
}

4. Astuces pour bien commencer

- Testez sur plusieurs appareils (ou utilisez les outils développeurs de Chrome).
- Pensez "Mobile First" : Commencez par le style mobile, puis ajoutez des règles pour les grands écrans.
- Utilisez des unités flexibles (%, vw, rem) plutôt que des pixels fixes (px).

5. Exemple complet

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemple Responsive</title>
  <style>
    /* Style par défaut (Mobile First) */
    body {
      font-family: Arial, sans-serif;
      margin: 0;
      padding: 10px;
      background-color: #f5f5f5;
    }
    
    .container {
      display: flex;
      flex-direction: column;
      gap: 10px;
    }
    
    .box {
      background-color: #4facfe;
      color: white;
      padding: 20px;
      border-radius: 8px;
    }
    
    /* Style pour tablettes (min 600px) */
    @media (min-width: 600px) {
      .container {
        flex-direction: row;
        flex-wrap: wrap;
      }
      
      .box {
        width: calc(50% - 10px);
      }
    }
    
    /* Style pour ordinateurs (min 992px) */
    @media (min-width: 992px) {
      .box {
        width: calc(33.33% - 10px);
      }
    }
  </style>
</head>
<body>
  <div class="container">
    <div class="box">Boîte 1</div>
    <div class="box">Boîte 2</div>
    <div class="box">Boîte 3</div>
  </div>
</body>
</html>

Résultat :
Mobile → Les boîtes sont empilées.
Tablette → 2 boîtes par ligne.
Ordinateur → 3 boîtes par ligne.

En résumé

Les Media Queries permettent d'adapter votre site à tous les écrans.
- Utilisez @media (max-width: ...) pour cibler les petits écrans.
- Testez sur plusieurs appareils avant de publier.

Maintenant, essayez d'ajouter un Media Query à votre site !
